## 2024-05-23 - [Empty State Welcome Panel]
**Learning:** Users running the tool without arguments are likely exploring. Replacing the standard `argparse` error with a visual "Welcome Panel" (using `rich`) transforms a "failure" state into a "discovery" moment, guiding them to successful first usage with copy-pasteable examples.
**Action:** Always consider the "no arguments" state as a landing page for CLI tools, not just an error condition.
